# WhatsApp NG

Это дополнение для NVDA, обеспечивающее улучшенную доступность веб-версии WhatsApp Desktop.

## Возможности

- **Alt+1**: Перейти к списку бесед WhatsApp
- **Alt+2**: Перейти к списку сообщений WhatsApp
- **Alt+D**: Перейти к полю ввода сообщения
- **Enter**: Воспроизвести голосовое сообщение (работает в личных чатах и группах)
- **Shift+Enter**: Открыть контекстное меню сообщения

### Переключение скриптов (изначально не назначены — настройте сочетания в «Жестах ввода»)

- Переключить фильтрацию телефонных номеров в списке чатов
- Переключить фильтрацию телефонных номеров в списке сообщений

## Требования

- NVDA 2021.1 или новее
- WhatsApp Desktop (веб-версия)

## Настройка

Фильтры телефонных номеров можно переключать:

- В списке чатов: Назначьте сочетание клавиш в диалоге «Жесты ввода»
- В списке сообщений: Назначьте сочетание клавиш в диалоге «Жесты ввода»
