# WhatsApp NG

Додаток NVDA, що забезпечує поліпшення доступності для веб-версії WhatsApp Desktop.

## Особливості

- **Alt+1**: перейти до списку бесід WhatsApp
- **Alt+2**: перейти до списку повідомлень WhatsApp
- **Alt+D**: перейти до редактора введення повідомлення
- **Enter**: відтворення голосового повідомлення (працює в особистих чатах і групах)
- **Shift+Enter**: відкрити контекстне меню повідомлення
- **Control+C**: копіювати поточне повідомлення в буфер обміну
- **Control+R**: прочитати повідомлення повністю (за необхідності натискає кнопку "читати далі")
- **Control+Shift+Enter**: Відповісти на повідомлення

### Скрипти перемикання (немає комбінації за замовчуванням — налаштуйте в «Жестах вводу»)

- Перемкнути фільтрацію телефонних номерів у списку розмов
- Перемкнути фільтрацію телефонних номерів у списку повідомлень
- Перемкнути автоматичний режим фокусування (дозволяє використовувати режим огляду за необхідності)

## Вимоги

- NVDA 2021.1 або новіша версія
- WhatsApp Desktop (веб-версія)

## Встановлення

1. Завантажте файл `whatsAppNG.nvda-addon`
2. У NVDA перейдіть до **Інструменти → Менеджер додатків**
3. Натисніть **Встановити** і виберіть файл
4. Перезавантажте NVDA

## Налаштування

Фільтри номерів телефонів можна перемикати:
- У списку розмов: Налаштуйте комбінацію у Жестах вводу
- У списку повідомлень: Налаштуйте комбінацію у Жестах вводу

Налаштуйте комбінації клавіш у:
**Меню NVDA → Налаштування → Жести вводу → WhatsApp NG**

## Журнал змін

### Версія 1.6.0 (2026-03-23)

**Додано:**
- Фільтрацію підказок використання: автоматично приховує підказки на кшталт "Для додаткових параметрів..." з оголошень повідомлень
  - Підтримує кілька мов
  - Можна перемикнути за допомогою NVDA+Shift+H

**Виправлено:**
- Подвійні оголошення в списку розмов: NVDA більше не оголошує кожен рядок чата двічі при навігації стрілками (#11)
- Навігація Alt+1: покращено надійність при пошуку списку розмов
- Копіювання повідомлень: покращено точність при копіюванні повідомлень за допомогою Control+C

### Версія 1.5.0 (2026-03-05)

- Ctrl+Shift+Enter: Відповісти на повідомлення (відкриває меню реакцій)

**Змінено:**
- Продуктивність значно оптимізована: Навігація тепер плавніша та чутливіша
