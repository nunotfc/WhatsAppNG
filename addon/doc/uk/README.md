# WhatsApp NG

Додаток NVDA, що забезпечує поліпшення доступності для веб-версії WhatsApp Desktop.

## Особливості

- **Alt+1**: перейти до списку бесід WhatsApp
- **Alt+2**: перейти до списку повідомлень WhatsApp
- **Alt+D**: перейти до редактора введення повідомлення
- **Enter**: відтворення голосового повідомлення (працює в особистих чатах і групах)
- **Shift+Enter**: відкрити контекстне меню повідомлення
- **Control+C**: копіювати поточне повідомлення в буфер обміну
- **Control+R**: прочитати повідомлення повністю (за необхідності натискає кнопку "читати далі")

### Скрипти перемикання (немає комбінації за замовчуванням — налаштуйте в «Жестах вводу»)

- Перемкнути фільтрацію телефонних номерів у списку розмов
- Перемкнути фільтрацію телефонних номерів у списку повідомлень
- Перемкнути автоматичний режим фокусування (дозволяє використовувати режим огляду за необхідності)

## Вимоги

- NVDA 2021.1 або новіша версія
- WhatsApp Desktop (веб-версія)

## Встановлення

1. Завантажте файл `whatsAppNG.nvda-addon`
2. У NVDA перейдіть до **Інструменти → Менеджер додатків**
3. Натисніть **Встановити** і виберіть файл
4. Перезавантажте NVDA

## Налаштування

Фільтри номерів телефонів можна перемикати:
- У списку розмов: Налаштуйте комбінацію у Жестах вводу
- У списку повідомлень: Налаштуйте комбінацію у Жестах вводу

Налаштуйте комбінації клавіш у:
**Меню NVDA → Налаштування → Жести вводу → WhatsApp NG**

## Журнал змін

### Версія 1.4.0 (2026-02-23)

**Додано:**
- Повну мовну підтримку: Арабська, Німецька, Іспанська, Італійська та Російська
- Оновлено український переклад з найновішими рядками

**Виправлено:**
- Помилку "Текст не знайдено" в Control+R після натискання кнопки "читати далі"
- Control+R тепер працює тільки з текстовими повідомленнями (показує "Не текстове повідомлення" для голосових/зображень)

**Змінено:**
- Посилання на репозиторій оновлено на новий репозиторій (nunotfc/WhatsAppNG)
- Документація: Всі локалізовані README тепер включають повний журнал змін до версії 1.3.0

### Версія 1.3.0 (2026-02-07)

**Додано:**
- Підтримку турецької мови
- Опцію перемикання автоматичного режиму фокусування (налаштуйте жест у Жестах вводу)

**Змінено:**
- Покращено продуктивність: команди навігації тепер швидші при повторному використанні
- Клавіша Escape тепер правильно передається в WhatsApp

**Виправлено:**
- Enter тепер відтворює відеоповідомлення (раніше працювало тільки для аудіо)

### Версія 1.1.1 (2025-01-31)

**Додано:**
- Control+R: Прочитати повідомлення повністю (автоматично натискає кнопку "читати далі")
- Control+C: Копіювати поточне повідомлення в буфер обміну
- Автоматичне вимкнення режиму огляду (зберігає режим фокусування активним для кращої роботи в WhatsApp)

**Змінено:**
- Покращено повідомлення про помилки: всі скрипти тепер надають чіткий відгук при збої
- Команди навігації (Alt+1, Alt+2, Alt+D) тепер мовчать при успіху
- Enter: Виявлення на основі повзуна замість підрахунку кнопок (більш надійно)

**Виправлено:**
- Alt+1 і Alt+2 правильно повідомляють про помилки, коли всі шляхи не вдаються
- Оптимізовано фільтрацію об'єктів для зменшення затримки вводу

### Версія 1.1.0 (2025-01-30)

**Додано:**
- Control+R: Прочитати повідомлення повністю
- Розумне відтворення голосових повідомлень з використанням виявлення повзуна

**Змінено:**
- Enter: Покращена логіка з використанням виявлення повзуна замість підрахунку кнопок

**Виправлено:**
- Alt+2 тепер правильно намагається використати всі шляхи навігації, якщо перша спроба не вдалася

### Версія 1.0.0 (2025-01-29)

**Перший випуск:**
- Комбінації клавіш навігації для списку бесід, списку повідомлень та поля написання повідомлень
- Відтворення голосових повідомлень з підтримкою особистих чатів і груп
- Доступ до контекстного меню для дій з повідомленнями
- Перемикання фільтрації номерів телефонів для бесід і повідомлень
- Автоматична активація режиму фокусування в WhatsApp Desktop
