# WhatsApp NG

Додаток NVDA, що забезпечує поліпшення доступності для веб-версії WhatsApp Desktop.

## особливості

- **Alt+1**: перейти до списку бесід WhatsApp
- **Alt+2**: перейти до списку повідомлень WhatsApp
- **Alt+D**: перейти до редактора введення повідомлення
- **Enter**: відтворення голосового повідомлення (працює в особистих чатах і групах)
- **Shift+Enter**: відкрити контекстне меню повідомлення

### Перемикання скриптів (немає початкової комбінації — налаштуйте в «Жестах вводу»)

- Перемкнути фільтрацію телефонних номерів у списку розмов
- Перемкнути фільтрацію телефонних номерів у списку повідомлень

## Вимоги

- NVDA 2021.1 або новіша версія
- WhatsApp Desktop (веб-версія)

## Конфігурація

Фільтри номерів телефонів можна перемикати:

- У списку розмов: Налаштуйте комбінацію у Жестах вводу
- У списку повідомлень: Налаштуйте комбінацію у Жестах вводу
